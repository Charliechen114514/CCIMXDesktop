如果您希望修改模块以便更好地移植到您的嵌入式设备，以下是供开发人员参考的模块描述。\hypertarget{md__components-explain-_c_n_autotoc_md1}{}\doxysection{\texorpdfstring{🔧 {\ttfamily app\+\_\+wrapper}}{🔧 {\ttfamily app\+\_\+wrapper}}}\label{md__components-explain-_c_n_autotoc_md1}

\DoxyHorRuler{0}
\hypertarget{md__components-explain-_c_n_autotoc_md3}{}\doxysection{\texorpdfstring{🧬 {\ttfamily builtin}}{🧬 {\ttfamily builtin}}}\label{md__components-explain-_c_n_autotoc_md3}


包含嵌入在桌面环境中的核心系统工具和默认实用程序。


\begin{DoxyItemize}
\item {\ttfamily core/icm20608}  {\bfseries{ICM-\/20608 IMU 传感器}} 的驱动程序和逻辑，用于处理陀螺仪和加速度计的实时数据采集，并通过 I2C 接口连接。（补充：如果您使用其他芯片，也应该修改并重写内部函数）
\item {\ttfamily gadgets}  界面友好的小工具，例如时钟、\+CPU/网络仪表或用于用户交互的快速启动图块。
\item {\ttfamily page}  用于内部 UI 路由的逻辑页面管理（例如，在主页、应用页面等之间切换）。
\item {\ttfamily sources/localweather}  用于获取或模拟本地天气数据的天气信息提供模块。
\item {\ttfamily sources/netcard}  用于处理网卡信息、状态显示，并可能提供 IP/网络诊断。
\item {\ttfamily ui}  内置模块特有的 UI 组件，这些文件支持桌面本身的 UI 样式。
\end{DoxyItemize}
