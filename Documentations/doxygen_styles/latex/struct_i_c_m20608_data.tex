\doxysubsection{Detailed Description}
Raw sensor data from \doxylink{class_i_c_m20608}{ICM20608} device. 

Data structure containing raw sensor readings from \doxylink{class_i_c_m20608}{ICM20608} device.

This structure holds 3D acceleration data, 3D rotation delta values, and temperature reading from the \doxylink{class_i_c_m20608}{ICM20608} sensor. All values are in raw sensor units and may require conversion to standard units.
