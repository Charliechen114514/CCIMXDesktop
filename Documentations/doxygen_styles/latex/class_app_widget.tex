\doxysubsection{Detailed Description}
The \doxylink{class_app_widget}{App\+Widget} class represents an application widget placed on the desktop. 

It displays an icon and app name, manages the underlying \doxylink{class_application_wrapper}{Application\+Wrapper}, and handles launching and status updates.
