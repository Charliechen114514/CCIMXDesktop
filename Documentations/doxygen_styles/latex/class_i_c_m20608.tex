\doxysubsection{Detailed Description}
Interface for \doxylink{class_i_c_m20608}{ICM20608} inertial measurement unit (IMU) sensor. 

Interface class to communicate with \doxylink{class_i_c_m20608}{ICM20608} sensor device.

This class provides platform-\/agnostic access to \doxylink{class_i_c_m20608}{ICM20608} sensor data, with implementations for both Windows and UNIX-\/like systems.
