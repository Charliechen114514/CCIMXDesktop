\doxysubsection{Detailed Description}
Handles and processes user input events for the game. 

The \doxylink{class_event_manager}{Event\+Manager} class processes keyboard and mouse events, translating them into game actions and indications. It acts as an intermediary between Qt\textquotesingle{}s input system and the game logic.
