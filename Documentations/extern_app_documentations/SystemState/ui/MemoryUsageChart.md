# **MemoryUsageChart**  

**头文件**: `MemoryUsageChart.h`  
**继承关系**: QWidget  
**简介**: 以饼图展示物理/虚拟内存使用情况的控件  

#### **公共槽函数**  

| 签名                                                  | 说明                   |
| ----------------------------------------------------- | ---------------------- |
| `flush_ui_once(const MemoryInfoFetcher::MemoryStat&)` | 更新内存数据并重绘图表 |

#### **核心组件**  

- 物理内存饼图（`physChart`）  
- 虚拟内存饼图（`virtChart`）  
- 标签自动更新机制  

#### **私有方法**  

| 方法              | 说明             |
| ----------------- | ---------------- |
| `update_charts()` | 更新指定饼图数据 |
| `update_labels()` | 同步更新文字标签 |
